# Part 1  
## Azure AI 語言服務於教育領域  
### 英文
1. 可以先給教學的講義資訊做為資料庫後，可以回答學生提出的問題。
2. 提供英文單字的例句 相似詞彙 修正文法
### 歷史  
1. 先給教學的範圍做為資料庫後，可以回答學生提出的問題。  
2. 包括歷史人物的生平、小故事  
3. 時間序、心智圖等....  
#### 同樣的各科都可以利用Azure AI的語言服務來幫助學生學習
